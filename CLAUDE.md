# Claude Code Projects Root
# This is the parent directory for all Claude Code projects

## Directory Structure
- `personal/` - Side projects and experiments
- `finance/` - Finance dashboard and budgeting tools
- `ai-tools/` - RAG system and AI utilities
- `learning/` - Tutorials and practice projects
- `templates/` - Reusable project starters

## Global Rules
All projects inherit from `~/.claude/CLAUDE.md`. Category-specific overrides are in each subfolder.

## Quick Start for New Projects
1. Create project in appropriate category folder
2. Initialize git: `git init`
3. Initialize Task Master: `task-master init`
4. Start with PRD: `/prd [project-name]`

## GitHub Integration
Every project should be a GitHub repository. Use `/ship` to commit, push, and create PRs.
