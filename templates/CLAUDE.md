# Project Templates
# Reusable starters for new projects

## Available Templates

### python-api/
FastAPI backend template with:
- Project structure
- Docker configuration
- Testing setup (pytest)
- Basic CRUD patterns

### react-app/
React frontend template with:
- Vite configuration
- TypeScript strict mode
- Tailwind CSS
- Component patterns

### fullstack/
Combined frontend + backend template

### exploration/
Minimal template for quick experiments

## Using Templates
Copy the template folder to your project category, then customize.

Example:
```bash
cp -r ~/claude-code/templates/python-api ~/claude-code/finance/my-new-project
cd ~/claude-code/finance/my-new-project
git init
task-master init
```
