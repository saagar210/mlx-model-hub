# Template Selection

[WARNING!] THIS IS AN EXPERIMENTAL TEMPLATE, DON'T USE IT UNLESS SPECIFICALLY REQUIRED!

Beautiful Reveal.js presentations with glass morphism design, runtime JSX compilation, and zero build steps.

Use when:
- Creating slide decks and presentations
- Showcasing products, features, or data
- Need elegant visual design with smooth animations
- Want runtime-editable presentations

Avoid when:
- Building web apps or dashboards
- Need forms, authentication, or CRUD operations
- SEO/static site generation required

Built with:
- Reveal.js 5 for presentations
- React 18 + Babel Standalone (runtime JSX compilation)
- Recharts for data visualization
- Tailwind CSS + glass morphism UI
- Pure CDN runtime (no build step)
- Custom component library (2 files: slides-library.jsx + slides-styles.css)
