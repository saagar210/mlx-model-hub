## KAS Integration

This project is connected to the Knowledge Activation System (KAS) for:
- Semantic search across your knowledge base
- Q&A with confidence scoring
- Capturing new learnings from this project

### Available MCP Tools

| Tool | Purpose |
|------|---------|
| `kas_search` | Search knowledge base with query |
| `kas_ask` | Ask questions, get AI-synthesized answers with citations |
| `kas_capture` | Save new learnings to KAS |
| `kas_stats` | Check KAS health and statistics |

### Usage Examples

```
# Search for relevant knowledge
kas_search: "FastAPI dependency injection patterns"

# Ask a question
kas_ask: "How do I implement streaming responses in FastAPI?"

# Capture a learning
kas_capture: title="Async Context Manager Pattern" content="..." namespace="projects/my-project"
```
