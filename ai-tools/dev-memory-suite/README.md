# Developer Memory & Context Suite

Capture everything you learn while coding. Make it searchable. Surface it exactly when needed.

## The Suite

| Tool | Description | Build Time |
|------|-------------|------------|
| **DevMemory** | Personal developer knowledge graph | 2-3 weeks |
| **CodeMCP** | MCP server for codebase understanding | 1-2 weeks |
| **ContextLens** | Smart context window manager | 1-2 weeks |

## Why This Exists

- Every developer needs this
- Leverages existing Knowledge Activation System
- Each tool feeds data to the others
- Natural fit with Claude Code usage

## Quick Start

```bash
# Start DevMemory (requires Knowledge System running)
cd devmemory
pip install -e .
devmemory serve
```
